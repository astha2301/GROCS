# ♻ GCROS — Garbage Collection Route Optimization System

GCROS (Garbage Collection Route Optimization System) is a full-stack smart waste management web application designed to optimize garbage collection routes in urban areas using Greedy and Travelling Salesman Problem (TSP) algorithms.

The system provides an interactive map interface where users can place garbage depots and bin locations, generate optimized collection paths, and simulate real-time waste collection operations.

---

# 🚀 Features

- Interactive map-based garbage management system
- Add and manage:
  - Garbage depots
  - Garbage bin points
- Route optimization using:
  - Greedy Algorithm
  - TSP (2-Opt Optimization)
- Dynamic route visualization on map
- Real-time collection simulation
- Driver management dashboard
- Truck fleet management system
- Distance and ETA calculation
- Responsive dark-themed UI
- Backend API integration using Express.js

🧠 Algorithms Used
Greedy Algorithm

The greedy nearest neighbor algorithm selects the closest unvisited bin point at every step to generate a quick route.

Travelling Salesman Problem (TSP)

TSP optimization improves the route generated by the greedy algorithm using the 2-Opt optimization technique to minimize overall travel distance.
